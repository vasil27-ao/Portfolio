Интеграция · Claude API · Trello API
Claude × Trello Integration
Claude API Trello API Node.js
Задача

Подготовка проектной доски Trello вручную занимает 2–3 часа: создание колонок, карточек, дедлайнов, ярлыков. Задача: автоматизировать этот процесс через текстовое описание проекта, чтобы менеджер мог разворачивать структуру за минуты, а не часы.

Реализация
  • Создание досок, колонок и карточек по текстовому запросу
  • Добавление описаний, дедлайнов и ярлыков через Claude API
  • Распределение задач по приоритетам и этапам
  • Готовая доска разработки бота-парсера — пример вывода интеграции
Результат Время подготовки проектной доски сокращено с 2–3 часов до 20 минут. Интеграция применена для планирования следующего проекта.

2
Автоматизация · Python · Telethon
Telegram Job Vacancy Filter Bot
Python Telethon python-telegram-bot
Задача

Ежедневный мониторинг вакансий в Telegram-каналах занимал несколько часов ручного просмотра. Нужен был инструмент, который отбирает релевантное автоматически без участия пользователя.

Проект спланирован и задокументирован с помощью Claude × Trello Integration: доска разработки создана до начала работ и использовалась на протяжении всего цикла.

Реализация
  • Парсинг сообщений из Telegram-каналов в реальном времени
  • Фильтрация по грейду, опыту, зарплате и формату работы
  • Кнопки управления: «в избранное» и «не интересно»
  • Статистика просмотров и найденных вакансий
  • Деплой с автозапуском через Windows Task Scheduler
Результат Бот запущен и работает: @myjobs_dzhen_bot. Время ежедневного поиска сокращено с нескольких часов до нескольких минут.

Методология · AI-assisted Development
Project Development Template
Markdown CLAUDE.md Claude Code .docx
Задача

При работе с AI-ассистентом каждая новая сессия начинается без контекста: модель не знает архитектуру, роли и бизнес-логику проекта. Нужна система, которая передаёт этот контекст структурированно — один раз, с первой же сессии.

Шаблон обобщает подход, опробованный на двух предыдущих проектах, и решает три задачи: быстрый старт с AI-ассистентом, передача проекта новому разработчику без потери контекста, и единый стандарт документации внутри команды.

Что входит
  • CLAUDE.md — контекстный файл проекта для AI-ассистента
  • project_template.docx — структура требований для PM
  • Инструкция: что заполнять и на каком этапе разработки
  • Staged-подход: файл растёт вместе с проектом
  • Адаптируется под мобайл, микросервисы, ML-продукты
Результат Шаблон проверен на двух проектах и опубликован как открытый инструмент для менеджеров и разработчиков, работающих с AI-ассистентами.
